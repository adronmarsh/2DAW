Evento change

Para relaizar este ejercicio se ha hecho uso del evento change.

Primero se creará un formulario con un input y un elemento p donde irá el texto modificado.

A continuación se creará en el script dos constantes que recojan los elementos creados anteriormente.

Por último se creará el addEventListener con el evento change.

Este evento contendrá una función que seleccionará el texto introducido y lo convertirá en mayúsculas mediante la función toUpperCase().
